
 Password Strength Analyzer with Custom Wordlist Generator

 📌 Description
This project analyzes password strength using the zxcvbn library and generates custom wordlists based on user inputs such as name, birth year, and pet name.

 🚀 Features
- Password strength analysis (0–4 score)
- Suggestion-based feedback
- Custom wordlist generation
- Leetspeak and pattern-based password creation
- GUI interface using Tkinter
- Export wordlist to .txt file

 🛠️ Technologies Used
- Python
- zxcvbn
- Tkinter

 📂 Files Included
- password_gui.py (Main GUI application)
- password_tool.py (CLI version)
- custom_wordlist.txt (Sample output)
- Project_Report.pdf

 🎯 Usage
Run the GUI using:
python password_gui.py

## 📸 Output
- GUI interface
- Generated wordlist file
